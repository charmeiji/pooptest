DOODLE IMAGES FOLDER
====================

This folder stores daily doodle images.

Naming format: YYYY-MM-DD.png or YYYY-MM-DD.jpg
Example: 2026-01-25.png

The calendar.html file looks for images in this folder.
It will try PNG format first, then JPG if PNG is not found.

Image specifications:
- Format: PNG or JPG
- Recommended width: 400px (as shown in calendar.html)
- Height: flexible
