Getting Started

2024-01-10

Setting up a minimal blog without any build tools or frameworks. Just plain HTML, CSS, and JavaScript.

The goal was to create something that:

  • Works as static files (no server-side code needed)
  • Is easy to update (just add files and edit JSON)
  • Loads quickly (minimal code, no dependencies)
  • Can be uploaded anywhere (Neocities, Nekoweb, GitHub Pages, etc.)

So far, it's working great! The simplicity makes it easy to maintain and customize.
